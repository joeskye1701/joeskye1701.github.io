Java递归生成XML

生成XML步骤：

  1. 生成根节点（root）；
  2. 判断root节点，是否有子节点，生成root节点子节点（二级节点）；
  3. 判断root节点的节点（二级节点）是否有子节点（三级节点）；
  4. 3这2步骤就可以写成递归，在写一个判断是否有子节点的函数和中文格式化。
